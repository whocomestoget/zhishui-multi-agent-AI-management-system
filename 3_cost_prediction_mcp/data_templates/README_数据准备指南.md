
# 智慧水电成本预测模型 - 数据准备指南（简化版）

## 概述

本指南帮助您准备用于训练智慧水电成本预测模型的数据。模型基于随机森林算法，采用**5个核心参数**进行成本预测，简化了数据收集难度，提高了模型的实用性。

## 文件说明

- `智慧水电成本预测数据模板.py`: Python格式的数据生成模板
- `智慧水电成本预测数据模板_YYYYMMDD.xlsx`: Excel格式的数据模板
- `数据说明文档_YYYYMMDD.json`: 详细的字段定义和验证规则
- `README_数据准备指南.md`: 本文件

## 数据要求

### 基本要求
- **样本数量**: 建议至少50个项目，理想情况下100+个项目
- **数据质量**: 确保数据准确、完整、一致
- **时间范围**: 建议使用近5-10年的项目数据
- **地理分布**: 覆盖不同地区和类型的项目

### 必填字段（简化版 - 仅6个字段）
1. **装机容量** (capacity_mw) - 单位：MW，范围：10-30000MW
2. **项目类型** (project_type) - 选项：常规大坝/抽水蓄能/径流式
3. **地理位置** (location_factor) - 选项：河谷/高山峡谷/山区
4. **建设周期** (construction_period) - 单位：年，范围：1-20年
5. **经济指标** (economic_indicator) - 开工年GDP增长率，范围：-1-1
6. **项目总投资** (total_cost) - 目标变量，单位：亿元

## 字段详细说明

### 1. 装机容量 (capacity_mw)
- **定义**: 水电站的总装机容量
- **单位**: MW（兆瓦）
- **范围**: 10-30000MW
- **示例**: 1000（表示1000MW）

### 2. 项目类型 (project_type)
- **定义**: 水电项目的智慧化程度分类
- **选项**: 
  - 常规大坝：传统大坝水电项目
- 抽水蓄能：抽水蓄能电站
- 径流式：径流式水电站
- **示例**: "抽水蓄能"

### 3. 地理位置 (location_factor)
- **定义**: 项目所在地的地形地貌特征
- **选项**: 河谷/高山峡谷/山区
- **影响**: 不同地形影响施工难度和成本
- **示例**: "河谷"

### 4. 建设周期 (construction_period)
- **定义**: 从开工到投产的建设时间
- **单位**: 年
- **范围**: 1-20年
- **示例**: 5（表示5年）

### 5. 经济指标 (economic_indicator)
- **定义**: 项目开工年的GDP增长率
- **单位**: 比例（如0.05表示5%，-0.02表示-2%）
- **范围**: -1-1（即-100%到100%）
- **示例**: 0.06（表示6%的GDP增长率）

### 6. 项目总投资 (total_cost)
- **定义**: 项目的总投资金额（目标预测变量）
- **单位**: 亿元
- **示例**: 50（表示50亿元）

## 数据收集建议

### 简化的数据来源
1. **装机容量**: 项目可行性研究报告
2. **项目类型**: 根据智能化设备配置判断
3. **地理位置**: 项目地址和地形图
4. **建设周期**: 项目计划或实际建设记录
5. **经济指标**: 国家统计局GDP数据
6. **总投资**: 项目决算或概算报告

### 数据质量控制
1. **成本数据统一**: 建议统一到2023年价格水平
2. **单位统一**: 严格按照上述单位要求填写
3. **逻辑检查**: 装机容量与总投资应有合理比例关系
4. **异常值处理**: 识别明显不合理的数据

## 使用步骤

1. **准备数据**: 收集上述6个字段的项目数据
2. **格式整理**: 按照字段要求整理数据格式
3. **质量检查**: 确保数据完整性和准确性
4. **模型训练**: 使用train_model.py进行模型训练
5. **预测测试**: 使用test_prediction.py验证模型效果

## 注意事项

⚠️ **重要提醒**:
- **装机容量**是最重要的特征，占预测权重90%以上
- **项目类型**影响智能化成本，请准确分类
- **地理位置**影响施工难度，不可忽略
- **建设周期**反映项目复杂度
- **经济指标**反映宏观经济环境对成本的影响
